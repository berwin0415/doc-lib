## 语言模型

什么是？ 打分的**方法**

### 用途
- 输入法
- 机器翻译
- 语音识别

### 发展历史：
- 传统（概率/统计）语言模型（没有上下文场景，数据集可能存在偏颇性，运算量过大）
- n-gram语音模型（n较小不准确，n较大时存储计算仍较大）
- 神经概率语言模型**NPLM**（**词向量**合理表示词与词之间的关系）（缺点 固定n大小）
- 循环神经网络**RNN**
