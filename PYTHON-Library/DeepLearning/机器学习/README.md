### 特征选择
特征选择就是单纯地从提取到的所有特征中选择部分特征作为训练集特征，
特征在选择前和选择后可以改变值、也不改变值，但是选择后的特征维数肯
定比选择前小，毕竟我们只选择了其中的一部分特征。

### 1. 为什么需要特征选择
- 冗余：部分特征的相关度高，容易消耗计算性能
- 噪声：部分特征对预测结果有负影响

### 2. 主要方法
- **Filter(过滤式):VarianceThreshold**
- **Embedded(嵌入式)：正则化、决策树**
- Wrapper(包裹式)
